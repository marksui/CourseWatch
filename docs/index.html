Built for students

Keep coursework visible without opening Canvas all day.

API

Canvas when available

Use a Canvas access token to fetch active courses and upcoming assignments.

ICS

Calendar Feed fallback

If your school blocks tokens, paste a Canvas Calendar Feed link and let CourseWatch read due dates from `.ics`.

+

External deadlines

Add exams, readings, club work, or anything Canvas misses. Manual deadlines live beside synced items.

24h

Local reminders

CourseWatch schedules macOS notifications 24 hours and 3 hours before due dates.

Setup choices

Start with the connection your school allows.

CourseWatch supports Canvas API tokens first, Calendar Feed when tokens are blocked, and manual external deadlines when Canvas cannot expose the item you need.

1

Paste your Canvas link in Settings.

2

Use a Canvas token, or switch to Calendar Feed and Auto Extract.

3

Click Test, Save, then let CourseWatch remind you.

Local-first

No backend server. No Canvas password.

Canvas tokens are stored in macOS Keychain.

Calendar Feed links and external deadlines are stored locally on your Mac.

CourseWatch is open source and provided as-is.
